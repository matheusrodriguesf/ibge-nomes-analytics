insights NomeBR Analytics

Esta aplicação é uma PoC desenvolvida com Angular 21, Angular Material e 3D Charts, utilizando dados de nomes brasileiros fornecidos pelo IBGE. O objetivo é oferecer uma interface intuitiva para explorar a popularidade dos nomes ao longo do tempo, permitindo que os usuários visualizem tendências e façam comparações entre diferentes nomes.

Nota: Esta é uma aplicação de demonstração.
